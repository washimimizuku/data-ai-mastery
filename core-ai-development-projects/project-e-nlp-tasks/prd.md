# PRD: NLP Multi-Task System

## Overview
Comprehensive NLP system covering classification, NER, summarization, QA, translation, and generation using Hugging Face models.

## Features
1. Text Classification (sentiment, topic)
2. Named Entity Recognition
3. Text Summarization
4. Question Answering
5. Translation
6. Text Generation
7. FastAPI endpoints
8. Model comparison

## Stack
Python, Hugging Face Transformers, spaCy, FastAPI
