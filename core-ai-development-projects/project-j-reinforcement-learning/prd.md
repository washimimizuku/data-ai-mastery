# PRD: Reinforcement Learning Agent

## Overview
Train RL agents using multiple algorithms (DQN, PPO, A2C) in Gymnasium environments.

## Features
1. Train agents (CartPole, LunarLander)
2. Multiple algorithms (DQN, PPO, A2C)
3. Custom environment
4. Training visualization
5. Agent evaluation
6. Model comparison
7. Deployment for inference

## Stack
Python, Gymnasium, Stable-Baselines3, PyTorch
