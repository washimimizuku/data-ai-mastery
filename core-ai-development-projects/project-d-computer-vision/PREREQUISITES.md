# Prerequisites: Computer Vision Pipeline Project

## Required Bootcamps

### 30 Days of Python for Data and AI
**Mandatory Lessons:**
- Day 1-5: Python Fundamentals
- Day 6-10: Data Structures & Control Flow
- Day 11-15: Functions & Modules
- Day 16-20: Object-Oriented Programming
- Day 21-25: Data Science Libraries (NumPy, Pandas)
- Day 26-30: Machine Learning with scikit-learn

### 100 Days of Data and AI
**Mandatory Lessons:**
- Day 1-5: Python for Data Science
- Day 6-10: NumPy & Pandas Fundamentals
- Day 11-15: Data Preprocessing & Feature Engineering
- Day 51-55: Machine Learning Fundamentals
- Day 61-65: Deep Learning Fundamentals
- Day 66-70: Neural Networks & PyTorch
- Day 71-75: Computer Vision & CNNs
- Day 76-80: Image Processing & OpenCV

**Nice to Have:**
- Day 56-60: Model Evaluation & Validation
- Day 81-85: Advanced Computer Vision
- Day 91-95: Model Deployment

## Estimated Prerequisites Time
- **Mandatory**: 55-65 hours
- **Nice to Have**: 15-20 hours
- **Total Recommended**: 70-85 hours

## Skills Validation
Before starting this project, ensure you can:
- Work with image data using OpenCV and PIL
- Implement CNN architectures with PyTorch
- Handle image preprocessing and augmentation
- Build object detection and classification pipelines
- Use pre-trained models and transfer learning
- Optimize models for inference performance
