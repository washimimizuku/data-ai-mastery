# PRD: Audio AI Pipeline

## Overview
Speech-to-text, text-to-speech, speaker diarization, and audio classification using open models.

## Features
1. Speech-to-text (Whisper)
2. Text-to-speech (Coqui TTS, Bark)
3. Speaker diarization
4. Audio classification
5. Voice cloning (basic)
6. Gradio interface
7. Real-time processing

## Stack
Python, Whisper, Coqui TTS, Bark, PyTorch
