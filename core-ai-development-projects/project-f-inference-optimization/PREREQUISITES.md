# Prerequisites: LLM Inference Optimization Project

## Required Bootcamps

### 30 Days of Python for Data and AI
**Mandatory Lessons:**
- Day 1-5: Python Fundamentals
- Day 6-10: Data Structures & Control Flow
- Day 11-15: Functions & Modules
- Day 16-20: Object-Oriented Programming
- Day 21-25: Data Science Libraries (NumPy, Pandas)

**Nice to Have:**
- Day 26-30: Machine Learning with scikit-learn

### 100 Days of Data and AI
**Mandatory Lessons:**
- Day 1-5: Python for Data Science
- Day 61-65: Deep Learning Fundamentals
- Day 66-70: Neural Networks & PyTorch
- Day 81-85: Large Language Models (LLMs)
- Day 91-95: Model Optimization & Quantization
- Day 96-100: Inference Acceleration Techniques

**Nice to Have:**
- Day 71-75: Natural Language Processing
- Day 86-90: Model Deployment & Serving
- Day 56-60: Performance Benchmarking

### 60 Days of Advanced Data and AI
**Mandatory Lessons:**
- Day 26-30: Advanced Model Deployment
- Day 31-35: Model Optimization Techniques
- Day 36-40: Hardware Acceleration (GPU, TPU)

**Nice to Have:**
- Day 41-45: Distributed Inference
- Day 46-50: Edge Deployment

## Estimated Prerequisites Time
- **Mandatory**: 50-60 hours
- **Nice to Have**: 20-25 hours
- **Total Recommended**: 70-85 hours

## Skills Validation
Before starting this project, ensure you can:
- Implement model quantization and pruning techniques
- Optimize inference performance with ONNX, TensorRT
- Handle batch processing and dynamic batching
- Implement caching and memory optimization
- Benchmark and profile model performance
- Deploy optimized models with FastAPI/TorchServe
