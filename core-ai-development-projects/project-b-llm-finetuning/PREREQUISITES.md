# Prerequisites: LLM Fine-Tuning Project

## Required Bootcamps

### 30 Days of Python for Data and AI
**Mandatory Lessons:**
- Day 1-5: Python Fundamentals
- Day 6-10: Data Structures & Control Flow
- Day 11-15: Functions & Modules
- Day 16-20: Object-Oriented Programming
- Day 21-25: Data Science Libraries (NumPy, Pandas)
- Day 26-30: Machine Learning with scikit-learn

### 100 Days of Data and AI
**Mandatory Lessons:**
- Day 1-5: Python for Data Science
- Day 11-15: Data Preprocessing & Feature Engineering
- Day 51-55: Machine Learning Fundamentals
- Day 61-65: Deep Learning Fundamentals
- Day 66-70: Neural Networks & PyTorch
- Day 71-75: Natural Language Processing
- Day 81-85: Large Language Models (LLMs)
- Day 91-95: Model Fine-Tuning & Transfer Learning
- Day 96-100: Advanced LLM Topics

**Nice to Have:**
- Day 6-10: NumPy & Pandas Fundamentals
- Day 56-60: Model Evaluation & Validation
- Day 76-80: Text Embeddings & Vector Databases

## Estimated Prerequisites Time
- **Mandatory**: 60-70 hours
- **Nice to Have**: 15-20 hours
- **Total Recommended**: 75-90 hours

## Skills Validation
Before starting this project, ensure you can:
- Work with PyTorch and Hugging Face Transformers
- Prepare and preprocess training datasets
- Implement fine-tuning strategies (LoRA, QLoRA, full fine-tuning)
- Monitor training metrics and handle overfitting
- Evaluate model performance on downstream tasks
- Deploy fine-tuned models for inference
