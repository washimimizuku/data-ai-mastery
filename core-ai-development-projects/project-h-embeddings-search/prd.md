# PRD: Embedding & Similarity Search

## Overview
Semantic search, clustering, and similarity algorithms using embeddings and vector databases.

## Features
1. Generate embeddings (multiple models)
2. Vector similarity search
3. Semantic search engine
4. Clustering (UMAP, t-SNE)
5. Duplicate detection
6. Recommendation system
7. FAISS vs Qdrant comparison
8. **Optional Rust**: Fast similarity implementations

## Stack
Python, sentence-transformers, FAISS, Qdrant, Rust (optional)
