# Prerequisites: Spark Streaming Demo Project

## Required Bootcamps

### 30 Days of Python for Data and AI
**Mandatory Lessons:**
- Day 1-5: Python Fundamentals
- Day 6-10: Data Structures & Control Flow
- Day 11-15: Functions & Modules
- Day 21-25: Data Science Libraries (NumPy, Pandas)

**Nice to Have:**
- Day 16-20: Object-Oriented Programming

### 100 Days of Data and AI
**Mandatory Lessons:**
- Day 1-5: Data Processing Fundamentals
- Day 21-25: Apache Kafka Fundamentals
- Day 26-30: Stream Processing with Kafka
- Day 31-35: Real-Time Data Pipelines
- Day 51-55: Apache Spark Fundamentals
- Day 56-60: Spark Streaming & Structured Streaming
- Day 61-65: Stream Processing Patterns

**Nice to Have:**
- Day 36-40: Data Quality & Validation
- Day 41-45: Change Data Capture (CDC)
- Day 71-75: Event-Driven Architecture

### 30 Days of SQL for Data and AI
**Mandatory Lessons:**
- Day 1-5: SQL Fundamentals
- Day 6-10: Advanced Queries
- Day 23-25: Real-Time Analytics

**Nice to Have:**
- Day 11-14: Data Manipulation
- Day 20-22: Performance Optimization

## Estimated Prerequisites Time
- **Mandatory**: 50-55 hours
- **Nice to Have**: 15-20 hours
- **Total Recommended**: 65-75 hours

## Skills Validation
Before starting this project, ensure you can:
- Set up and configure Apache Spark
- Work with Spark DataFrames and SQL
- Implement Structured Streaming applications
- Connect Spark to Kafka for real-time processing
- Handle streaming data transformations and aggregations
- Manage streaming checkpoints and fault tolerance
- Monitor and troubleshoot streaming applications
