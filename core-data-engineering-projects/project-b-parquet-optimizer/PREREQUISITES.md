- **Mandatory**: 30-35 hours

## Skills Validation
Before starting this project, ensure you can:
- Work with Parquet files using Python (pyarrow, pandas)
- Understand columnar storage concepts
- Implement data compression and optimization techniques
- Benchmark and measure performance improvements
- Analyze storage efficiency and query performance
