# Prerequisites: Rust Data Parser Benchmark Project

## Required Bootcamps

### 30 Days of Rust for Data and AI
**Mandatory Lessons:**
- Day 1-5: Rust Fundamentals & Syntax
- Day 6-10: Ownership, Borrowing & Lifetimes
- Day 11-15: Data Structures & Collections
- Day 16-20: Error Handling & Pattern Matching
- Day 21-25: File I/O & Data Processing
- Day 26-30: Performance Optimization & Benchmarking

**Nice to Have:**
- All lessons are recommended for comprehensive Rust knowledge

### 100 Days of Data and AI
**Mandatory Lessons:**
- Day 1-5: Data Processing Fundamentals
- Day 16-20: File Formats & Serialization
- Day 56-60: Performance Benchmarking

**Nice to Have:**
- Day 6-10: Data Structures & Algorithms
- Day 11-15: Memory Management

## Estimated Prerequisites Time
- **Mandatory**: 35-40 hours
- **Nice to Have**: 10-15 hours
- **Total Recommended**: 45-55 hours

## Skills Validation
Before starting this project, ensure you can:
- Write efficient Rust code with proper memory management
- Handle file I/O operations in Rust
- Implement data parsing and serialization
- Use Rust benchmarking tools (criterion.rs)
- Optimize code for performance
- Handle errors gracefully in Rust
