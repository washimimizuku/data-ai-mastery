# Environmental Data & AI Projects

Focused, standalone environmental science projects showcasing AI/ML applications for conservation biology, climate science, and environmental monitoring. All projects run locally using open-source tools and publicly available environmental datasets.

## Projects Overview

### Project A: Satellite Imagery Forest Change Detection
**Time**: 2-3 days | **Focus**: Computer vision for deforestation monitoring

Build a system to detect forest changes using satellite imagery and computer vision models.

**Stack**: Python, OpenCV, scikit-image, Sentinel-2 data, GDAL, Rasterio

---

### Project B: Species Audio Classification Pipeline
**Time**: 2-3 days | **Focus**: Bioacoustics ML for wildlife monitoring

Create an ML pipeline to identify bird/animal species from audio recordings.

**Stack**: Python, librosa, PyTorch, Hugging Face transformers, scikit-learn

---

### Project C: Climate Data Time Series Forecasting
**Time**: 2-3 days | **Focus**: Environmental time series prediction

Build forecasting models for temperature, precipitation, and extreme weather events.

**Stack**: Python, Prophet, LSTM, pandas, plotly, NOAA climate data

---

### Project D: Carbon Footprint Calculator with LLM
**Time**: 2-3 days | **Focus**: GenAI for environmental impact assessment

Create an intelligent carbon footprint calculator using LLMs for natural language queries.

**Stack**: Python, Ollama, LangChain, FastAPI, Streamlit

---

### Project E: Environmental Sensor Data Streaming
**Time**: 2-3 days | **Focus**: IoT data processing for environmental monitoring

Build a real-time environmental monitoring system processing sensor data streams.

**Stack**: Python, Kafka, InfluxDB, Grafana, MQTT, pandas

---

## Getting Started

1. **Prerequisites**: Python 3.9+, Docker, Git
2. **Setup**: Each project has its own virtual environment and requirements
3. **Data**: All projects use publicly available environmental datasets
4. **Documentation**: Each project includes detailed README and notebooks

## Learning Path

- **Beginner**: Start with Project C (Climate Forecasting) - uses familiar time series concepts
- **Intermediate**: Move to Project A (Satellite Imagery) or Project B (Audio Classification)
- **Advanced**: Tackle Project D (LLM Integration) or Project E (Streaming)
