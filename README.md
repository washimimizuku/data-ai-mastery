# Data & AI Mastery

A complete learning path from basic programming to advanced data and AI engineering, with hands-on bootcamps and production-ready projects.

## Goal

Master data and AI engineering through structured bootcamps and real-world projects, transforming from a basic developer into an advanced data and AI engineer.

## Time Investment

- Core Languages: 60 hours (SQL + Python) - optional
- 100 Days Bootcamp: 100 hours
- Quick Win Projects: 40 hours (4-6 projects)
- 60 Days Advanced: 60 hours
- 2 Comprehensive Projects: 320 hours (160h each)
- **Total**: 580 hours (3 hours/day)

### Standard Path (9 months)
- Core Languages: 90 hours (SQL + Python + Rust) - optional
- 100 Days Bootcamp: 100 hours
- Quick Win Projects: 60 hours (6-8 projects)
- 60 Days Advanced: 60 hours
- 3 Comprehensive Projects: 600 hours (200h each)
- **Total**: 910 hours (3.5 hours/day)
