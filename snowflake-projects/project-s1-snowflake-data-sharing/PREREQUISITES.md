# Prerequisites: Enterprise Data Sharing Project

## Required Bootcamps

### 30 Days of Snowflake for Data and AI
**Mandatory Lessons:**
- Day 1: Snowflake Architecture & Account Setup
- Day 2: Virtual Warehouses & Compute Management
- Day 3: Databases, Schemas & Tables
- Day 4: Data Loading with Snowpipe
- Day 5: Data Sharing Fundamentals
- Day 6: Secure Data Sharing
- Day 7: Data Marketplace & Provider Setup
- Day 8: Cross-Cloud Data Sharing
- Day 15: Role-Based Access Control (RBAC)
- Day 16: Data Governance & Classification
- Day 17: Account Usage & Monitoring
- Day 25: Multi-Account Architecture

**Nice to Have:**
- Day 9: Time Travel & Fail-Safe
- Day 10: Zero-Copy Cloning
- Day 18: Resource Monitors & Cost Control
- Day 19: Performance Optimization

### 30 Days of SQL for Data and AI
**Mandatory Lessons:**
- Day 1-5: SQL Fundamentals (SELECT, WHERE, JOIN)
- Day 6-10: Advanced Queries (Subqueries, CTEs, Window Functions)
- Day 15-17: Data Security & Access Control
- Day 20-22: Performance Optimization

**Nice to Have:**
- Day 11-14: Data Manipulation (INSERT, UPDATE, DELETE)
- Day 18-19: Stored Procedures & Functions

## Estimated Prerequisites Time
- **Mandatory**: 35-40 hours
- **Nice to Have**: 15-20 hours
- **Total Recommended**: 50-60 hours

## Skills Validation
Before starting this project, ensure you can:
- Set up Snowflake accounts and databases
- Configure secure data sharing between accounts
- Implement RBAC and data governance policies
- Monitor data sharing usage and costs
- Write complex SQL queries for data analysis
